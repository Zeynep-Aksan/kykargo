# kykargo
KYKARGO, üniversite öğrencilerinin kargo teslimat süreçlerini dijitalleştiren bir yardımlaşma sistemidir. Kullanıcılar okul numaralarıyla hızlıca sisteme dahil olup, kargo yardımı isteyebilir veya başkalarının kargosunu alarak süreci yönetebilir. PHP backend ve modern CSS arayüzü ile geliştirilmiştir.
